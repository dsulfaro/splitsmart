# API Endpoints

## HTML API

### ROOT
- `GET /` - loads splash page if not logged/signed in

## JSON API

### USERS
- `POST /api/users` - create user

### SESSIONS
- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### SIGNUP
- `GET /api/signup` - render signup page

### FRIENDS
- `POST /api/friends/:id`
- `DELETE /api/friends/:id`
- `GET /api/friends/:id`
